מזהה החשבון שלך
-
הדבק את המזהה הזה או את קוד ההטמעה באתר הלקוח שלך. כל מוצר שתוסיף כאן יישאב לשם אוטומטית.
לכל חשבון נוצר קוד קצר של שורה-שתיים בלבד. כל שאר הנתונים נשאבים אוטומטית מ-Firebase.

                  
אפשר להטמיע כ-iframe מיידי בלי להעתיק קוד ארוך.

                

                

                  
שתי אפשרויות חיבור ברורות

                  

                    1. עיצוב ידני מתוך המערכת
                    
השתמשו ב-iframe המוכן. המערכת מציגה את הכרטיסים, את כפתור הסל ואת כל העיצוב שאתם מגדירים כאן.

                  

                  

                    2. עיצוב לפי הקוד של אתר הלקוח
                    
שלבו את הנתונים בלבד: האתר של הלקוח שולט ב-CSS/HTML שלו, והמערכת הזאת רק מספקת מוצרים, מחיר, תיאור, תוספות ותמונות.

                  

                  
איך לחבר באתר רגיל

                  
    
                    
  1. העתיקו את מזהה החשבון או את קוד ה-iframe מתוך המסך הזה.
    2. 
                    
  2. להטמעה מהירה: הדביקו את ה-iframe בכל אזור HTML באתר.
    3. 
                    
  3. להטמעת נתונים בלבד: העתיקו את אובייקט הנתונים והלבישו עליו את עיצוב האתר הקיים.
    4. 
                    
  4. בחרו לכל מוצר אם להשתמש בעיצוב האתר או בעיצוב מותאם מתוך המערכת.

הגדרות המערכת

                
שליטה בשפה, בהעדפות הממשק ובמצב האימות.

              

            

            

              

                
מצב רב-לשוני
כפתורי השפה פעילים בכל המסכים.

                
הצגת מבצעים מודגשת
מציג תגי מבצע בצורה בולטת בכרטיסים ובהטמעה.

                
התאמה למובייל
הרכיבים אורגנו מחדש למובייל ולדסקטופ בצורה נוחה יותר.

                
Google Auth חלק
Popup עם Redirect fallback וטיפול בשגיאות נפוצות.

              

              

                
מצב המערכת

                

                  
• הנתונים נשמרים ב-Firestore תחת המשתמש המחובר.

                  
• כל תמונה נשמרת ב-Firebase Storage עם שיוך למוצר.

                  
• קוד ההטמעה משתמש במזהה החשבון בלבד.

                  
• ניתן לייבא את הנתונים לכל אתר שתומך ב-HTML מותאם.

                  
• העיצוב נשמר בנפרד כדי לשרת גם את ההטמעה.

                  
• המבנה מותאם להפרדת נתונים בטוחה לפי כללי Firebase.
